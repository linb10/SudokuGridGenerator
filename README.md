# SudokuGridGenerator


Sudoku Grid Generator

Brandon Lin

This program generates a fully random Sudoku grid, completed and solved.

How to run the program: Download the .java file and import it to a Java Console complier (Eg: CodeHS)

Summary: The Sudoku grid is generated in a 3 step process. First, 3 diagonal 3x3 boxes are generated with a random list of integers 1-9, as each of the boxes are indepenent of each other. Second, the rest of the grid is filled in row by row while checking for integer validity by Sudoku rules. Third, the grid outline and integer values of the double array is printed.
